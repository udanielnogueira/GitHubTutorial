+++++++++++++
GITHUB ONLINE
+++++++++++++

Ir para o seu perfil: PROFILE ICON > YOUR PROFILE

Criar novo repositório: Em seu perfil > REPOSITORIES > NEW

Clonar um repositório: Vá para o repositório > FORK

Ver código e ver histórico: Clique no repositório > Estará disponível cada arquivo com seu respectivo histórico

++++++++++++++
GITHUB DESKTOP
++++++++++++++

Clonar um repositório: CURRENT REPOSITORY> ADD > CLONE REPOSITORY

Alterar e fazer o push: Altere > Vá para CHANGES > COMMIT TO MAIN > PUSH ORIGIN (Ctrl + P)

Captar da nuvem: REPOSITORY > PULL

Pull Request: BRANCH > CREATE PULL REQUE
